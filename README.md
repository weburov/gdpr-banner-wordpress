# GDPR Банер за WordPress без плъгин

Този проект показва как да добавите GDPR банер на вашия WordPress сайт без да използвате плъгин.

## Инструкции
1. Качи `gdpr-banner.php` във вашата тема или child theme.
2. Увери се, че структурата на папките `css/` и `js/` е запазена.
3. Банерът ще се появи автоматично и ще изчезне след като потребителят го затвори.
